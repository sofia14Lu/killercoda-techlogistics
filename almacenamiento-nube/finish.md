# Conclusión

En este laboratorio se implementó una solución de almacenamiento cloud para TechLogistics S.A.

## Lo aprendido

- Uso de almacenamiento de objetos compatible con S3
- Configuración de buckets y acceso mediante AWS CLI
- Automatización de backups con scripts Bash
- Importancia de la escalabilidad y recuperación de datos
- Reducción de carga en almacenamiento local

## Beneficios obtenidos

- Mayor disponibilidad de archivos
- Protección frente a fallos físicos
- Automatización de procesos IT
- Optimización de costos de almacenamiento
