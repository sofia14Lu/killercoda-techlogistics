# Conclusión

En este laboratorio se implementó una solución de almacenamiento cloud utilizando MinIO y herramientas compatibles con AWS S3.

## Conceptos aprendidos

- Diferencia entre almacenamiento de bloques y almacenamiento de objetos
- Uso de AWS CLI para gestión cloud
- Automatización de backups y migraciones
- Importancia de la escalabilidad y disponibilidad de datos

## Resultado

TechLogistics S.A. ahora cuenta con una solución más escalable, automatizada y segura para la gestión de archivos y respaldos.
